open terminal
navigate to project

'cd path/to/project'

if you are working on other computers - do this first!

`git pull`

`git clone <paste link from github here>`

`git add .`

after important changes

`git commit -am'useful commit message'`

to upload to github

`git push origin main`

`git push`
